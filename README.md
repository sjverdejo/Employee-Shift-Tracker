# Employee-Shift-Tracker

A full stack application for my employer to track employee hours. Created in TypeScript using React, Express, Node, PostgreSQL managed by Supabase. Unit tests written using Jest library to ensure code quality and functionality. Using Tailwind CSS library for visually appealing and responsive web design. Employers can create, view, update, and delete new accounts, admin (managers) or non (employees). Employers can create, view, update, and delete shifts.
